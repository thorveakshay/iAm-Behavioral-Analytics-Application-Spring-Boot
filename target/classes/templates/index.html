App Details

I want to build something which will help people analyze, review themselves. Find out their personality type using some unique questions and possibly algorithms. Suggest connecting similar minded persons. Also generate some personalized tips to improve, refer some commercial books/products to them. Also, try to show some relevant news and articles to them using google feeds etc. Basically, make them love and focus on themselves.

Built-with

Java 8, Maven, MongoDB, Spring 5, Spring Boot, Mustache Templating, HTML, CSS and JS

Secured with Facebook OAuth2

Use have to login using their facebook account. No spam and personalised application. Optional organizational login using Okta also integrated.

Continuous Integration and Delivery (CI/CD) Pipeline

GitHub and Heroku. Next intigration with AWS,S3 and Amazon CloudWatch‎.

Social integrations/h4>

Twitter API, Okta API and Facebook API
